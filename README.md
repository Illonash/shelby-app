# 📂 Shelby App: Smart Decentralized File Storage

**Shelby App** is a next-generation decentralized file storage platform that merges blockchain security with artificial intelligence. Designed with a sleek, modern aesthetic, Shelby ensures your data remains secure, private, and effortlessly managed through a smart, user-centric interface.

## Why Shelby?

Unlike traditional cloud storage, Shelby gives you full sovereignty over your data using Web3 technology, supported by an intelligent assistant that guides you through every step of the process.

### Key Features
* **Decentralized Security:** Seamless integration with the **Aptos Wallet** for secure, password-less authentication and on-chain verification.
* **Shelby AI Assistant:** A built-in smart assistant that helps manage your files, provides real-time security insights, and responds to user commands.
* **Live Intelligence Status:** A dynamic "Pulse" indicator that monitors network activity and encryption integrity in real-time.
* **Aesthetic & Modern UI:** A premium user experience featuring a signature **Soft Pink & Purple** glassmorphism theme, designed for maximum clarity and comfort.

## 🛠️ Tech Stack
- **Frontend:** React.js + TypeScript
- **State Management:** React Hooks
- **Styling:** Custom CSS3 (Advanced Animations & Glassmorphism)
- **Blockchain:** Aptos Network Integration
- **Identity:** Official Shelby Branding
